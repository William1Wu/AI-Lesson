1）特徵值（eigen value）與特徵向量（eigen vector）各別定義
存在一非 0 向量 v，使方陣 A 滿足 Av = λv（其中 λ 為純量），則稱 v 為 A 的特徵向量 v，λ 為 A 的特徵值。
特徵向量及特徵值在幾何上的意義：
如果某矩陣對某一或某些向量的作用只有伸縮變換、而無改變方向的效果（或只變成負方向），那麼這些向量就稱為這個矩陣的特徵向量，而伸縮的比例就是特徵值。

2）奇異值與特徵值關係
　<1> 給定一大小為 M x M 的方陣 A，其對角化分解可以寫成 A = UΛU^(-1)，
　　　其中 U 的每一列都是特徵向量，而 Λ 對角線上的元素是從大到小排列的特徵值。
　<2> 當方陣 A 是一對稱矩陣時，則存在一個對稱對角化分解 A = QΛQ^T（Q^T 代表 Q 的轉置矩陣），
　　　其中 Q 的每一列都是相互正交的特徵向量，且是單位向量，而 Λ 對角線上的元素是從大到小排列的特徵值。
　<3> 當給定一大小為 M x N 的矩陣 A，雖然矩陣 A 不一定是方陣，但大小為 M x M 的 AA^T 與 N x N 的 A^TA 卻是對稱矩陣，
　　　若 AA^T = P(Λ1)P^T、A^TA = Q(Λ2)Q^T，則矩陣 A 的奇異值分解為 A = PΣQ^T，
　　　其中 P 是 M x M 矩陣，其每一列都是 AA^T 的特徵向量，又稱為 A 的左奇異向量（left singular vector），
　　　Q 是 N x N 矩陣，其每一列都是 A^TA 的特徵向量，又稱為 A 的右奇異向量，
　　　Λ1 是 M x M 矩陣、Λ2 是 N x N 矩陣，兩矩陣對角線上的非零元素相同（即矩陣 AA^T 和矩陣 A^TA 的非零特徵值相同），
　　　而 Σ 是 M x N 矩陣，其位於對角線上的元素便稱為奇異值（singular value）
　<4> 給定一個很大的 M x N 矩陣 A，若矩陣 A 的秩 k 遠小於 M 和 N，我們只需儲存 k(M + N + 1) 個元素就能得到原矩陣，
　　　即 k 個奇異值、個 kM 左奇異向量的元素、和 kN 個右奇異向量的元素。
　　　若採用一個秩 r 矩陣去逼近，則只需儲存更少的 r(M + N + 1) 個元素。因此奇異值分解是一種重要的數據壓縮方法。
